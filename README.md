# WC2026 Predictions Platform V15

Αλλαγές σε σχέση με V14:

- Προστέθηκε fallback φόρτωσης του φύλλου «ΑΠΟΤΕΛΕΣΜΑΤΑ» με βάση το όνομα του sheet, ώστε να μη σπάει η σελίδα αν αλλάξει το gid.
- Η σελίδα πλέον διαβάζει και τη στήλη G του «ΑΠΟΤΕΛΕΣΜΑΤΑ», όπου καταχωρίζονται οι ομάδες που προκρίθηκαν στους 32, στους 16 κ.λπ.
- Όταν δεν υπάρχουν πλέον upcoming ματς ομίλων, εμφανίζει μήνυμα ότι η φάση των ομίλων ολοκληρώθηκε.
- Το Match Center μπορεί να δείξει πρόσφατες καταχωρίσεις προκρίσεων από τη στήλη G, αν δεν υπάρχουν διαθέσιμα αποτελέσματα αγώνων.

Αρχεία προς ανέβασμα στο GitHub: `index.html`, `app.js`, `style.css`.
