# WC2026 Predictions Platform — V16

Έκδοση για τη μετάβαση από τη φάση των ομίλων στα νοκ άουτ.

## Τι άλλαξε

- Το section `Upcoming Picks` μετονομάστηκε σε `Προκρίσεις`.
- Αν δεν υπάρχουν πλέον επόμενα ματς ομίλων, η σελίδα δεν δείχνει κενό μήνυμα.
- Διαβάζει τις πραγματικές προκρίσεις από τη στήλη `G` του φύλλου `ΑΠΟΤΕΛΕΣΜΑΤΑ`.
- Εμφανίζει πόσοι παίκτες είχαν προβλέψει κάθε ομάδα που πέρασε στους 16.
- Αν προχωρήσει η διοργάνωση, το ίδιο section μπορεί να δείξει και πιο προχωρημένο στάδιο όταν συμπληρωθεί η στήλη G.
- Διορθώθηκε η ανάγνωση των προβλέψεων `Στους 16` ώστε να βασίζεται στις γραμμές 40–55 και να μην επηρεάζεται από ελληνικά/λατινικά roman labels.

## Αρχεία προς ανέβασμα

Ανέβασε στο GitHub repository τα εξής αρχεία:
